Description

RPG game engine implementation.

Features

  • Maps can be edited with tiled editor
  • Support for basic lighting, directional and point lights.
  • Support and dynamic maps (Try walking to first floor)
  • Enemies will follow you, but can be killed using space bar.

Compatibility

  • Desktop: Needs a webgl enabled browser. https://get.webgl.org/
  • Phone/tablet: Some support for touchscreen control. Needs WebGL.
